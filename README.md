# 📊 BI com Excel: Trabalhando com Tabelas Dinâmicas e Power Pivot

Projeto desenvolvido durante o curso **"BI com Excel: Trabalhando com Tabelas Dinâmicas com Power Pivot"** da Alura, com foco na criação de análises de dados utilizando Tabelas Dinâmicas, Segmentação de Dados, Linha do Tempo, Modelagem de Dados e linguagem DAX dentro do Excel.

##  Objetivo do Projeto

Desenvolver relatórios analíticos utilizando recursos avançados do Excel para transformar dados em informações estratégicas, permitindo:

* Resumir grandes volumes de dados com Tabelas Dinâmicas;
* Aplicar filtros avançados através de Segmentação de Dados;
* Realizar análises temporais com Linha do Tempo;
* Construir modelos relacionais utilizando Power Pivot;
* Criar colunas calculadas utilizando DAX;
* Gerar análises dinâmicas para apoio à tomada de decisão.

## 📋 Conceitos Trabalhados

Durante o curso foram explorados os principais recursos de Business Intelligence disponíveis no Excel:

### Tabelas Dinâmicas

* Criação de relatórios dinâmicos
* Organização de campos
* Agrupamento de informações
* Resumo e análise de dados
* Personalização e formatação de relatórios

### Segmentação de Dados e Linha do Tempo

* Criação de filtros visuais interativos
* Filtragem por categorias
* Filtragem por períodos de tempo
* Navegação dinâmica entre informações

### Power Pivot

* Habilitação e configuração do suplemento
* Criação de relacionamentos entre tabelas
* Construção de modelo de dados
* Aplicação do conceito de Tabela Fato e Tabela Dimensão
* Gerenciamento de relacionamentos
